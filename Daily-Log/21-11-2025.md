# Take Aways
### How to approach a solution
1. Translate the English to Math
    - Convert the given data into variables
    - Form an equation with the given relations
2. Simplify the equation aggressively
3. Check feasibility
4. Pick one variable to iterate, solve for the other
    - Free one variable
    - Express the other
    - Apply constraints
    - Count the valid ones
5. Spot the pattern
